HOW IT WORKS:
-------------

PRIMARY SIGNAL: Head Pose Estimation
- Uses MediaPipe Face Mesh + solvePnP
- Calculates yaw, pitch, roll angles
- Focused if: |yaw| ≤ 30°, |pitch| ≤ 25°, |roll| ≤ 35°
- Only used if pose confidence > 0.3

FALLBACK SIGNAL: Image Quality
- Used when head pose fails or confidence < 0.3
- Analyzes sharpness, brightness, contrast
- Conservative threshold: quality > 0.4 for focused
- Lower confidence score (0.5x) to indicate fallback

TEMPORAL SMOOTHING:
- Requires 3 consecutive distraction frames before marking unfocused
- Prevents flickering from single bad frames
- Provides stable results over time
