================================================================================
🎯 WHAT IT DOES
================================================================================

1. Loops through all folders in students/
2. For each image:
   - Preprocesses (resize, normalize)
   - Runs through CNN model
   - Predicts engagement class
3. Computes weighted score
4. Determines status: Focused / Neutral / Distracted
5. Displays in beautiful dashboard

================================================================================
📊 ENGAGEMENT CLASSES
================================================================================

Engaged         (+2 points)  ✅
Confused        (+1 point)   🤔
Frustrated      (0 points)   😤
Looking Away    (-1 point)   👀
Bored           (-1 point)   😴
Drowsy          (-2 points)  💤

================================================================================
🐛 TROUBLESHOOTING
================================================================================

Model not found:
- Ensure best_model_state.bin is in project root

No students found:
- Run tracking system first to collect images
- Check students/ directory exists

Port in use:
- Kill process: lsof -ti:5000 | xargs kill -9
- Or use different port: python api_server.py --port 5001

Out of memory:
- Reduce batch size in engagement_analytics.py

================================================================================
📊 SCORING SYSTEM
================================================================================

For each student:
1. Count predictions for each class
2. Apply weights: Engaged(+2), Confused(+1), Frustrated(0), 
                  Looking Away(-1), Bored(-1), Drowsy(-2)
3. Calculate total score
4. Normalize to 0-100 scale
5. Determine status:
   - Score ≥ 70% → Focused
   - Score 40-69% → Neutral
   - Score < 40% → Distracted
