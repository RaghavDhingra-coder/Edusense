================================================================================
🔧 COMMON ADJUSTMENTS
================================================================================

TOO MANY DUPLICATE IDs?
Edit config_robust.py:
  REID_SIMILARITY_THRESHOLD = 0.50  # Lower
  REID_COOLDOWN_PERIOD = 5.0        # Increase

DIFFERENT STUDENTS MERGED?
Edit config_robust.py:
  REID_SIMILARITY_THRESHOLD = 0.60  # Raise
  REID_QUALITY_THRESHOLD = 40.0     # Raise

TOO SLOW?
  python3 main_robust.py --preset fast

Or edit config_robust.py:
  SKIP_FRAMES = 3
  INFERENCE_SIZE = 416

================================================================================
🎯 KEY FEATURES
================================================================================

1. HYBRID MATCHING
   - Embedding similarity (50%)
   - Spatial proximity (30%)
   - Temporal continuity (20%)

2. QUALITY ASSESSMENT
   - Size quality (30 points)
   - Confidence quality (30 points)
   - Sharpness quality (25 points)
   - Aspect ratio quality (15 points)
   - Threshold: 30/100

3. TEMPORAL CONSISTENCY
   - Cooldown: 3 seconds before new ID
   - Grace period: 5 seconds for lost tracks
   - Automatic track recovery

4. EMBEDDING SMOOTHING
   - Rolling average (up to 10 embeddings)
   - Outlier rejection (similarity < 0.4)
   - Stable matching

5. POST-PROCESSING
   - Duplicate detection (similarity >= 0.65)
   - Mixed detection (DBSCAN clustering)
   - Automatic merge/split/clean

🐛 TROUBLESHOOTING
================================================================================

NO FACES DETECTED?
- Check lighting
- Lower CONFIDENCE_THRESHOLD to 0.3
- Verify camera works

IDs KEEP CHANGING?
- Increase REID_COOLDOWN_PERIOD to 5.0
- Lower REID_SIMILARITY_THRESHOLD to 0.50

SYSTEM TOO SLOW?
- Use --preset fast
- Increase SKIP_FRAMES to 3
- Use GPU: --device cuda:0

TOO MANY FOLDERS?
- Run: python3 folder_cleanup.py --action merge
