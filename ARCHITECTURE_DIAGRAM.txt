================================================================================
  🎯 FILTERING PIPELINE (Multi-Layer)
================================================================================

Detection
    │
    ├─> Layer 1: Confidence Check
    │   └─> confidence ≥ 0.6 ?
    │       ├─> YES: Continue
    │       └─> NO: Reject
    │
    ├─> Layer 2: Size Check
    │   └─> width ≥ 40 AND height ≥ 40 ?
    │       ├─> YES: Continue
    │       └─> NO: Reject
    │
    ├─> Layer 3: Aspect Ratio Check
    │   └─> 0.5 ≤ (width/height) ≤ 1.5 ?
    │       ├─> YES: Continue
    │       └─> NO: Reject
    │
    ├─> Layer 4: Time Interval Check
    │   └─> 1 second passed since last save ?
    │       ├─> YES: Continue
    │       └─> NO: Skip save
    │
    ├─> Layer 5: Crop Validation
    │   └─> Valid crop dimensions ?
    │       ├─> YES: Continue
    │       └─> NO: Reject
    │
    └─> Layer 6: Final Validation
        └─> Crop aspect ratio valid ?
            ├─> YES: SAVE
            └─> NO: Reject

Result: Only high-quality face crops are saved

================================================================================
  🚀 PERFORMANCE OPTIMIZATION
================================================================================

GPU Path (Fast):
    Frame → YOLOv8-Face (FP16) → ByteTrack → Crop → Save
    Speed: 15-20 FPS
    Device: CUDA

CPU Path (Slower):
    Frame → YOLOv8-Face (FP32) → ByteTrack → Crop → Save
    Speed: 3-5 FPS
    Device: CPU

Optimizations:
    ✅ FP16 half precision (2x speed on GPU)
    ✅ Efficient filtering (early rejection)
    ✅ Time-based saving (avoid redundant I/O)
    ✅ Optimized inference size (1280 for accuracy)
