  📊 EXPECTED BEHAVIOR

SCENARIO 1: New Student Appears
  → Face detected
  → ByteTrack assigns Track ID
  → ReID creates new Student ID
  → Console: "✨ New student: Student 1 (Track 1)"
  → Saves to students/student_1/

SCENARIO 2: Student Changes Seat
  → Student moves
  → ByteTrack loses old track
  → New track detected
  → ReID extracts embedding
  → Matches with existing student
  → Console: "🔄 ReID: Track 5 matched to Student 1 (similarity: 0.850)"
  → Continues saving to students/student_1/

SCENARIO 3: Temporary Occlusion
  → Student looks down
  → ByteTrack keeps track alive (60 frame buffer)
  → Student looks up
  → Same track ID maintained
  → Same Student ID maintained
  → No ReID needed

SCENARIO 4: Brief Disappearance
  → Student leaves frame
  → ByteTrack buffer keeps track for 60 frames
  → Student returns
  → Track recovered
  → Student ID maintained

  REID_EXTRACT_INTERVAL = 30  # Extract embedding every 30 frames
  REID_LOW_CONFIDENCE_THRESHOLD = 0.7  # Re-extract if confidence < 0.7

================================================================================
  🔧 TROUBLESHOOTING
================================================================================

ISSUE: "Failed to load InsightFace"
  SOLUTION: pip install insightface onnxruntime

ISSUE: Slow performance
  SOLUTIONS:
    1. Lower inference size: INFERENCE_SIZE = 640
    2. Increase cache duration: REID_EMBEDDING_CACHE_DURATION = 10.0
    3. Increase extraction interval: REID_EXTRACT_INTERVAL = 60

ISSUE: Too many false matches
  SOLUTION: Increase similarity threshold
    REID_SIMILARITY_THRESHOLD = 0.7

ISSUE: Not enough matches (too many new students)
  SOLUTION: Decrease similarity threshold
    REID_SIMILARITY_THRESHOLD = 0.5

ISSUE: IDs still changing frequently
  SOLUTIONS:
    1. Increase track buffer: TRACK_BUFFER = 90
    2. Lower track threshold: TRACK_LOW_THRESH = 0.2
    3. Increase max age: MAX_AGE = 90
